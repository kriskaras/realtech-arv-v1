# Scope (V1)
- Deterministic comp retrieval + scoring
- Baseline ARV + confidence range
- Optional: Human-in-the-loop comp curation (Keep/Reject) with confidence penalty
- AI is narration only (no AI comp selection)

- Deployed link works end-to-end
- API returns contract-shaped JSON
- UI shows baseline ARV + range + confidence
- UI shows candidate comps + similarity breakdown
- Curate comps updates ARV + confidence (if implemented)
